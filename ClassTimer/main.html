Information

Window Modegrid_view

Window mode allows you to create a smaller window to put the schedule in. The benefits of this mode is that you can switch between windows to quickly access the schedule. To switch between windows quickly, press alt + tab. If you hold down alt + tab, you can see and switch between all open windows (cycle using tab). Quikly pressing alt + tab will switch you between your last opened window (in most cases, the schedule window).

Brower Tab Modeopen_in_new

Browser Tab Mode will open the window in the current tab. This is the same as the Window mode, but in the current browser window. The benefits to this mode are that the title of the web page will change, depending on the current period and time left.
